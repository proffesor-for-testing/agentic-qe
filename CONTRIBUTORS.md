## How to Contribute

We welcome contributions of all kinds! Here's how you can help:

### Getting Started

1. **Read the Guidelines**: Check our [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions
2. **Find an Issue**: Look for issues labeled `good first issue` or `help wanted`
3. **Fork & Clone**: Fork the repo and create your feature branch
4. **Make Changes**: Implement your changes with tests
5. **Submit PR**: Open a pull request with a clear description

### Types of Contributions

- **Code**: Bug fixes, new features, optimizations
- **Documentation**: Guides, API docs, examples
- **Testing**: Unit tests, integration tests, e2e tests
- **Design**: UI improvements, accessibility
- **Community**: Answer questions, review PRs, mentoring

### Recognition

When you contribute, you'll be:
- Added to this contributors list
- Credited in release notes
- Awarded contribution badges
