**Title:** Validate flywheel promotions with a DoE/ANOVA screen before treating them as compounding wins

**Body:**

The self-learning flywheel (ADR-176) + signed RVFA propagation (ADR-177) are a great
substrate. One methodological suggestion from a parallel effort (the agentic-qe
learning-integrity work, ADR-117–122), offered in case it's useful:

**The concern.** ADR-176 reports "2 compounding promotions," but (per the shipped
notes) the **human anchor was held flat, not improved** — the gains are measured on
a proxy metric. That's exactly the shape where a promotion can be **noise dressed as
signal**: a retrieval/policy tweak that moves a proxy without moving the thing you
actually care about.

**A cautionary precedent from our side.** We ran a structurally similar
retrieval-tuning mechanism through a **Design-of-Experiments / ANOVA screen** (our
ADR-122) and found the effect was **"beads"** — not statistically distinguishable
from noise once we controlled for fixture variance. Same headline shape ("it
improved!"), null result underneath. We only caught it because we screened it.

**The offer.** Our DoE harness is a small, transferable pattern you could apply to
the flywheel's promotion decision:

1. **Frozen anchor (ADR-117 analog):** a fixed, versioned evaluation set the flywheel
   can't tune against, so "improvement" is measured against a target that doesn't move.
2. **ANOVA screen (ADR-122):** before a promotion is called real, run an F-test across
   fixtures — is the candidate-vs-baseline delta significant, or within fixture noise?
   Only promote on a significant result; otherwise report "no effect" honestly.
3. **Honest nulls:** a promotion that fails the screen is a *finding*, not a failure —
   it stops a noise tweak from compounding into the propagated config (ADR-177), where
   a false promotion would otherwise entrench itself.

Happy to share our ANOVA-screen implementation / walk through the "beads" case if
it's useful. The goal is just to make sure the flywheel's compounding is compounding
*signal*, not compounding *drift*.
