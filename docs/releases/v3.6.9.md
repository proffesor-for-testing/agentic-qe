# v3.6.9 Release Notes

**Release Date:** 2026-02-17

## Highlights

Major code quality and reliability release — eliminates all unsafe type casts, closes the self-learning feedback loop, wires 10+ persistence gaps, and applies P0 security hardening across the platform.

## Fixed

- **ESM TypeScript import failure in devcontainers** (#267) — Build scripts now handle ESM resolution correctly in devcontainer environments.
- **DomainServiceRegistry not registered in kernel** — Domain services are now discoverable at runtime via the kernel bootstrap.
- **ONNX adapter using wrong vectors table** — Embeddings now route to the unified `vectors` table.
- **10 empty persistence tables** — Wired storage for mincut health monitor, strange loop, code-intelligence index, embedding cache, GOAP planner, and others. Fixed async race in handler factory.
- **Self-learning feedback loop not closing** (#265) — Learning outcomes now feed back into the system to improve future test generations.
- **Queen coordinator missing tasksCompleted counter** — Swarm progress tracking now works correctly.

## Changed

- **Eliminated all 77 `as any` type casts** — Proper typed interfaces across 20 files.
- **P0 security and performance fixes** — JWT validation, input sanitization, safe JSON parsing from QE fleet analysis.
- **Code quality overhaul** — 41 improvement goals across 6 phases, covering all 13 domain coordinators.
- **God file splits** — Queen coordinator and large coordinators split into focused modules.
- **Consolidated .db gitignore** — Unified database file exclusions.
