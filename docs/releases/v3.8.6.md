# v3.8.6 Release Notes

**Release Date:** 2026-03-23

## Highlights

Security ReDoS fixes, flaky timer elimination, and large module decomposition for maintainability.

## Fixed

- **4 CodeQL ReDoS alerts resolved** — Eliminated all `js/polynomial-redos` high-severity vulnerabilities in security validators (command-validator, input-sanitizer, path-traversal-validator, regex-safety-validator).
- **58 real setTimeout calls eliminated** — Replaced with `vi.useFakeTimers()` in 5 flaky test files for deterministic execution.
- **time-crystal.ts decomposed** — Split 1,714-line module into 6 focused modules (max 447 lines) with backward-compatible re-exports.
- **qe-reasoning-bank.ts decomposed** — Split 1,941-line module into 7 focused modules (max 738 lines) with backward-compatible re-exports.
