# v3.7.23 Release Notes

**Release Date:** 2026-03-16

## Highlights

This is the largest feature release since v3.7.0 — the RuVector integration brings native vector search (150x faster), intelligent model routing that learns to cut costs, and safety gates that validate agent outputs before they execute. All new features ship behind feature flags for safe adoption.

## What's New for Users

### Faster Search, Lower Memory
- **Native HNSW vector search** replaces the pure-JS implementation. Pattern searches that took seconds now complete in single-digit milliseconds.
- **4x memory reduction** via temporal tensor compression — your agent's learned patterns take 75% less RAM.
- **HNSW health monitoring** with spectral analysis alerts you when indexes need rebalancing.

### Smarter Agents
- **Neural model routing (TinyDancer)** learns which tasks need expensive models and which can use fast/cheap ones. Over time, this reduces inference costs while maintaining quality.
- **Cross-domain transfer learning** — insights from API testing improve UI testing and vice versa. New projects benefit from patterns learned on previous ones.
- **DAG attention scheduler** runs independent tests in parallel while respecting dependencies, cutting suite execution time.
- **Regret tracker** detects when routing decisions degrade over time and auto-corrects.

### Safer Agents
- **Coherence-gated actions** — a 3-filter safety pipeline validates every agent output before execution, catching hallucinated tool calls and broken reasoning chains.
- **Witness chain audit trail** — every agent decision is SHA-256 hash-linked for full traceability. Know exactly why an agent made each choice.
- **Reasoning QEC** — majority-vote consensus across multiple reasoning paths for higher confidence decisions.

### New Capabilities
- **Behavior tree orchestration** — compose complex agent workflows with Sequence, Selector, and Parallel nodes with built-in retry and fallback.
- **CNN visual regression** — detect visual UI changes using spatial pooling embeddings instead of brittle pixel matching.
- **Cognitive container export/import** — save and share an agent's full learned state as a portable RVF v2 snapshot.
- **QE dashboard** — browser-based pattern explorer with cluster visualizations and WASM-accelerated search.

### Safe Adoption
All 15 new capabilities ship behind feature flags (default off). Enable incrementally:
```bash
aqe ruvector enable native-hnsw
aqe ruvector enable neural-routing
aqe ruvector enable coherence-gate
aqe ruvector profile list
```

## Stability

- 4 flaky tests fixed (shared singleton contamination under full-suite contention)
- ARM64 install failure resolved (`@ruvector/tiny-dancer-linux-arm64-gnu` moved to optionalDependencies)
- 26 new regression tests covering all new code paths
- Obsolete v2-to-v3-migration test removed (source was already gone in v3.7.22)

## Getting Started

```bash
npx agentic-qe init --auto
```

To enable RuVector features after upgrading:
```bash
aqe ruvector status        # Check available features
aqe ruvector enable --all  # Enable all (or pick individually)
aqe audit coherence        # Verify safety gates work
```
