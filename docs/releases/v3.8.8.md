# v3.8.8 Release Notes

**Release Date:** 2026-03-24

## Highlights

AQE agents and skills now work without an MCP server. All 431 MCP tool references have been migrated to CLI commands, with a new `aqe memory` command providing full memory operations via the terminal.

## Added

- **`aqe memory` CLI command** with 7 subcommands: `store`, `get`, `search`, `list`, `delete`, `share`, `usage`. Full parity with MCP memory tools, supporting `--json` output for scripting.
- **Web tree-sitter WASM parsers** for Python, Java, C#, Rust, and Swift — cross-language code intelligence without native binary dependencies.
- **`--with-mcp` init flag** — MCP server configuration is now opt-in. Run `aqe init --with-mcp` to include it, or add manually with `claude mcp add aqe -- aqe-mcp`.

## Changed

- **MCP dependency removed from all agent/skill definitions** — 230+ files migrated from `mcp__agentic-qe__*` tool calls to `aqe` CLI equivalents across `.claude/`, `assets/`, and `.kiro/` directories.
- **18 QE skill descriptions improved** — Third-person voice, concrete trigger terms (framework names, technique keywords), and "Use when..." clauses for better skill activation.
- **15 QE skill names standardized** to kebab-case matching directory names.
