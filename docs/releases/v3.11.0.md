# v3.11.0 Release Notes

**Release Date:** 2026-06-20

## Highlights

Run test generation on **free local models first** (opt-in) — a cheap-first path
with an automatic repair loop that escalates to paid models only for the hard
cases, so most routine generation can run at $0. Inspired by the "the harness
matters as much as the model" economics from MetaHarness Darwin Mode.

## Added

- **Free-tier local test generation (opt-in).** Set `AQE_FREE_TIER=1` (or
  `enableFreeTier` in coordinator config) to try a free/local model before any
  paid LLM call. Default model `qwen3:8b`. See the
  [Free-Tier Local Models guide](../guides/free-tier-local-models.md).
- **Configurable providers** — local Ollama, cloud Ollama, OpenRouter free
  models, or any OpenAI-compatible endpoint (vLLM, LM Studio, Groq, …). Keys are
  read from named env vars and never stored.
- **Repair loop** — invalid first output is fed back and retried before fallback.
- **Automatic escalation** — hard cases escalate up the existing model ladder
  (Haiku → Sonnet → Opus) via the router.
- **Self-learning routing feedback** — free-tier outcomes feed the routing
  calibrator and escalation tracker, lifting model-tier confidence over time.
- **`agentic-qe/routing/free-tier` package export** for programmatic config.

## Fixed

- Restored `@ruvector/ruvllm-*` optional platform packages dropped from the
  lockfile by a Dependabot update (could break `npm ci`). (#541)
- Multi-file test-generation requests fall back to the full path so every file is
  covered (the free-tier path handles the single-file case).
- Security bump of the transitive `hono` dev dependency. (#541)

## Changed

- Auto-escalation tracker is now ladder-configurable (generic over tier names) so
  a free local tier can sit below the paid tiers. Fully backward compatible.
- **QE framework evolved PACT → PACTS** — adds a fifth principle, **Structured**
  (governance, observability, and explainability of agent behavior; measure
  *confidence*, not trust), inspired by DORA's research on AI-assisted delivery.
  Ships with a playbook + readiness-assessment guide. The skill id/dir
  `holistic-testing-pact` and schema slugs are kept stable for cross-repo
  compatibility; the output schema gains an optional, non-breaking `structured`
  dimension. PACT originated with Reuven Cohen (Agentics Foundation). (#545)
