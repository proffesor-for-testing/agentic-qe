# v3.8.7 Release Notes

**Release Date:** 2026-03-23

## Highlights

Unified hypergraph persistence into memory.db, added CLI and MCP query tools for the code knowledge hypergraph.

## Added

- **Hypergraph CLI commands** — New `aqe hypergraph` subcommands (`stats`, `untested`, `impacted`, `gaps`) for querying the code knowledge hypergraph directly from the terminal.
- **Hypergraph MCP tool** — `hypergraph_query` MCP tool exposes the same queries to AI agents, scoped to 5 agent roles.
- **Code index extractor** — Shared async batched entity extractor for functions, classes, interfaces, arrow functions, and method definitions with 12 unit tests.
- **Shell completions** — Bash, Zsh, Fish, and PowerShell completion support for hypergraph subcommands.

## Fixed

- **Unified hypergraph persistence** — Eliminated separate `hypergraph.db` file; all hypergraph data now lives in the unified `memory.db`.
- **Stale governance shard paths** — Updated 12 governance shards + constitution from `v3/src/domains/` to `src/domains/`.
- **Init phase 06 hypergraph bootstrap** — `aqe init --auto` now builds hypergraph tables during initialization.
- **Connection leak in CLI handler** — Added `ensureInitialized` guard and proper cleanup.
- **HypergraphDegraded event** — Coordinator publishes degradation events on init failure for observability.
