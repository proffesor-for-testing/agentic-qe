# v3.11.3 Release Notes

**Release Date:** 2026-06-27

## Highlights

Tests and evals you can **actually trust**. AQE now grades a generated test by
*running* it — against the real code (it must pass) and against deliberately-broken
versions (it must catch them) — instead of checking for keywords. Test generation
leads with **durable tests** (invariants, contracts, properties) that survive a
rewrite, quality reports gain a **mutation-score / regenerability** signal next to
coverage, and a new **conservation guard** keeps AQE's own commands and output
formats stable for your scripts. Plus: pick a **cheaper or local model** for test
generation and prove it clears the oracle. See the
[oracle-evals plan](../plans/oracle-evals-durable-tests-plan.md) (ADR-113),
[conservation-layer policy](../guides/conservation-layer-policy.md) (ADR-114), and
[cheaper-model guide](../guides/cheaper-model-eval-lanes.md).

## Added

- **Oracle evals** — skill evals run the generated test against a reference
  implementation plus seeded mutants; an assertion-less or wrong test is rejected.
- **Durable-first test generation** — `qe-test-generation` emits invariants,
  contracts, and property-based tests first (with a "would this survive a rewrite?"
  language-swap check), keeping happy-path/mock tests as a labeled, disposable tier.
- **Regenerability gate** — `qe-quality-assessment` reports mutation score + a
  per-module regenerability score alongside coverage (warn by default, opt-in blocking).
- **Conservation guard** — `npm run verify:conservation` + `verify:skill-parity` catch
  breaking changes to CLI commands, report output schemas, the dashboard API, and skill
  trees, with a deprecation policy so removals are deliberate and announced.
- **Cheaper model lanes** — `npm run providers:health`, `eval:models`, `eval:live:claude|ollama`
  to choose and validate a budget/local model for test generation.

## Fixed

- **OpenRouter provider** now routes through the provider manager (it previously failed
  because it identified itself as `openai` instead of `openrouter`).
- **Gemini provider** default model updated to `gemini-2.5-flash` (the old default was
  retired by Google); supported-model list and pricing refreshed from the live API.

## Changed

- `qe-test-generation` and `qe-quality-assessment` skill guidance reframed around
  *regenerability* (durable tests + fault detection), synced across the shipped skill trees.

## Notes

Additive and bug-fix only — no breaking changes to the CLI or published API. Internal
tooling (oracle/gate/conservation) ships as npm scripts + libraries.
