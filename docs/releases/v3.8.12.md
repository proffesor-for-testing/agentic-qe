# v3.8.12 Release Notes

**Release Date:** 2026-03-29

## Highlights

RuVector Phase 5 (ADR-087) delivers pattern intelligence, graph learning, and scale optimization across 4 milestones with 456 new tests.

## Added

- **HDC Pattern Fingerprinting** — 10K-bit binary hypervectors with O(1) XOR-based similarity search, wired into PatternStore as Hamming-distance pre-filter.
- **CUSUM Drift Detection** — Two-sided statistical change-point detection across all 4 coherence gate types (retrieve/write/learn/act), auto-enabled via feature flag.
- **Delta Event Sourcing** — Pattern version history with rollback via SQLite `pattern_deltas` table, wired into PatternStore store/recordUsage paths.
- **EWC++ Three-Loop Activation** — instantAdapt/recordOutcome/backgroundConsolidate protocol wired into quality-assessment, test-generation, and contract-testing coordinators.
- **GraphMAE Self-Supervised Learning** — SPSA optimizer for masked graph autoencoders with decoder head reconstruction (SCE loss). Wired into coordinator-gnn.
- **Modern Hopfield Networks** — Exponential-capacity exact pattern recall with L2-normalized patterns for consistent softmax attention. Wired into PatternStore store/search.
- **Cold-Tier GNN Training** — LRU-cached mini-batch trainer with memory cap enforcement and file-backed larger-than-RAM graphs.
- **Meta-Learning Enhancements** — DecayingBeta, PlateauDetector with CUSUM, ParetoFront, and CuriosityBonus wired into DomainTransferEngine.
- **PageRank Pattern Importance** — Citation graph pipeline with blended scoring in pattern-promotion.
- **Spectral Graph Sparsification** — Degree-based leverage heuristic wired into mincut-wrapper as optional preprocessing.
- **Reservoir Replay Buffer** — Coherence-gated admission with CUSUM drift detection in ExperienceReplay.
- **E-prop Online Learning** — 12 bytes/synapse eligibility traces with feedback alignment (RL algorithm #10).
- **Granger Causality** — VAR + F-test for test failure prediction, wired into defect-intelligence coordinator.
- **456 new tests** across 11 test files with performance benchmarks.

## Fixed

- **HDC pre-filter dropping search results** — The R1 HDC fingerprint pre-filter was eliminating candidates with low Hamming similarity instead of just reordering them, causing text searches to return empty results.

## Changed

- Coherence gate refactored from 930-line monolith into 4 focused modules under 500 lines each.
- `RLAlgorithmType` extended from 9 to 10 members (added `eprop`).
- Xorshift128 PRNG extracted to `src/shared/utils/xorshift128.ts` for cross-module reuse.
- ADR-087 promoted from Proposed to Active.

## Security

- Bumped `path-to-regexp` from 0.1.12 to 0.1.13 (dependabot).
