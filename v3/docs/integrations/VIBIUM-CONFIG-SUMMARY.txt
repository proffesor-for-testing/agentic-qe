6. QUICK START COMMANDS
================================================================================

Initialize Fleet with Vibium:
await agentic_qe__fleet_init({
  enabledDomains: [
    'test-generation', 'test-execution', 'coverage-analysis',
    'quality-assessment', 'visual-testing'
  ],
  topology: 'hierarchical',
  maxAgents: 15
});

Check Fleet Status:
const status = await agentic_qe__fleet_status({ verbose: true });
console.log(status.mcp.servers); // Should include vibium

Spawn Visual Agent:
const agent = await agentic_qe__agent_spawn({
  domain: 'visual-testing',
  type: 'specialist'
});

Run Visual Test:
const result = await vibiumClient.visualRegression({
  baseline: 'baseline-name',
  url: 'https://app.example.com',
  threshold: 0.99
});

Store Result:
await agentic_qe__memory_store({
  key: 'visual-test-result-' + timestamp,
  namespace: 'visual-testing',
  value: result
});

7. ENVIRONMENT VARIABLES
================================================================================

Development Mode:
VIBIUM_HEADLESS=false      # Show browser window
VIBIUM_DEBUG=true          # Enable debug logging
VIBIUM_TIMEOUT=60000       # 60 second timeout
NODE_NO_WARNINGS=1         # Suppress warnings

Production/CI Mode:
VIBIUM_HEADLESS=true       # Run headless (faster)
VIBIUM_DEBUG=false         # No debug output
VIBIUM_TIMEOUT=30000       # 30 second timeout
NODE_NO_WARNINGS=1
NODE_OPTIONS=--max-old-space-size=4096

Custom Configuration:
VIBIUM_PORT=9999           # Specify port (auto-assign if not set)
VIBIUM_CHROME_ARGS="..."   # Custom Chrome flags
HTTPS_PROXY=proxy:8080     # Corporate proxy support

10. TROUBLESHOOTING
================================================================================

Vibium Server Not Starting:
→ Check: cat /workspaces/agentic-qe/.claude/mcp.json | jq '.mcpServers.vibium'
→ Verify: npx -y vibium --version
→ Fix: npm cache clean --force && npx -y vibium@latest

Chrome Not Found:
→ Install: npx @vibium/cli install-chrome
→ Requires: 2GB+ disk space, internet connection

Port Already in Use:
→ Set: VIBIUM_PORT=9999 in environment
→ Or: Let it auto-assign by leaving PORT unset

Memory Issues in CI/CD:
→ Set: NODE_OPTIONS=--max-old-space-size=4096
→ Consider: Reduce parallel agents from 5 to 2-3

MCP Configuration Invalid:
→ Validate: cat /workspaces/agentic-qe/.claude/mcp.json | jq .
→ Expected: Valid JSON with vibium server defined
