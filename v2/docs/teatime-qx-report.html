üìê QX Methodology

What is Quality Experience (QX)?

QX = Quality + UX is a holistic framework that combines Quality Advocacy (ensuring fitness for purpose, value delivery) with User Experience (usability, accessibility, delight). Unlike traditional UX which focuses solely on end users, QX recognizes that quality is co-created by all stakeholders.

QX Analysis Dimensions

  • User Experience (UX): Usability, accessibility, performance, aesthetics
  • Quality Advocacy: Value delivery, fitness for purpose, ethical standards
  • Community Value: Knowledge sharing, professional development, networking
  • Stakeholder Co-creation: How all parties contribute to and benefit from quality

Evaluation Criteria

Each dimension is scored on a 10-point scale based on: effectiveness (does it work?), efficiency (how well does it work?), satisfaction (do stakeholders value it?), and impact (does it advance the profession?).
